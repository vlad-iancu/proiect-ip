## Installation
Create a virtual environment `python3 -m venv ./venv` \
`source ./venv/bin/activate` \
`pip3 install -r requirements.txt` \
Create the `conf.json` file (see [conf.sample.json](./conf.sample.json)) \
`./run.sh` \
or \
`run.ps1` if you are on Windows
