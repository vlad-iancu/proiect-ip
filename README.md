# proiect-ip
Proiect la Ingineria Programării

## Installation
`pip3 install -r requirements.txt` \
Create the `conf.json` file (see [conf.sample.json](./conf.sample.json))
`./run.sh` \
or \
`run.ps1` if you are on Windows
